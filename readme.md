# Hera

> Phần mềm quản lí kì thi Tin học. 
> Hiện đang trong giai đoạn thử nghiệm.


## Dev
> Dependencies: 
> 	nodejs v6
> 	npm with `browserify`, `bower` and `gulp` installed

```
$ npm install
$ bower install
$ gulp build
```

### Run

```
$ npm start
```
